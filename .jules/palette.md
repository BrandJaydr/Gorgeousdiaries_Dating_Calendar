## 2025-05-14 - Interaction Feedback for Background Actions
**Learning:** Background actions like file downloads (e.g., ICS calendar export) are often visually "silent" within the application UI, relying on browser-level notifications that users might miss. Providing immediate, in-place visual feedback significantly improves the perceived responsiveness of the app.
**Action:** Implement a 2-second "Success" state (e.g., changing button text to "Added!" and swapping to a checkmark icon) for all background export/download actions.
